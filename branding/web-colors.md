# Paleta de colores – Web corporativa

Este documento define la paleta de colores oficial para la página web
corporativa de la empresa.

La paleta fue diseñada para transmitir:
- Confianza
- Entorno industrial
- Profesionalismo B2B

## Color principal (estructura)

Uso:
- Header
- Footer
- Títulos principales
- Fondos oscuros

RGB: 34, 51, 72  
HEX: #223348

## Color secundario (equilibrio)

Uso:
- Fondos claros
- Secciones
- Separadores
- Cards

RGB: 205, 208, 210  
HEX: #CDD0D2

## Color de acento (acciones)

Uso:
- Botones
- Enlaces importantes
- Hover

RGB: 235, 103, 38  
HEX: #EB6726

Este color debe usarse con moderación.

Texto principal:  
RGB: 33, 33, 33  
HEX: #212121

Fondo blanco:  
RGB: 255, 255, 255  
HEX: #FFFFFF

### Costos aproximados (en pesos colombianos - COP)
- Diseño de paleta de colores: $500,000 - $1,200,000 COP (utilizando herramientas gratuitas como Canva)
- Implementación en la web: $1,000,000 - $2,500,000 COP (freelancers o pequeñas agencias locales)

### Tecnologías utilizadas
- Herramientas de diseño: Canva (opción gratuita), Adobe XD, Sketch
- Frameworks CSS: Tailwind CSS, Bootstrap, Bulma (alternativa ligera)

### Certificaciones recomendadas
- Cursos básicos en diseño de interfaces (UI) en plataformas como Platzi o Udemy
- Certificación en experiencia de usuario (UX) (opcional para proyectos pequeños)
