# Plantilla de cotización ejecutiva

Este documento define la estructura estándar de las cotizaciones
enviadas a clientes empresariales.

1. Información del cliente
   - Nombre de la empresa
   - Persona de contacto
   - Datos de contacto (correo, teléfono)

2. Descripción del servicio
   - Breve resumen del servicio ofrecido
   - Beneficios clave para el cliente

3. Alcance del trabajo
   - Detallar las actividades incluidas
   - Especificar entregables

4. Materiales incluidos
   - Listar materiales necesarios (si aplica)
   - Indicar si los costos están incluidos en el precio

5. Tiempo estimado
   - Duración del proyecto
   - Fechas tentativas de inicio y finalización

6. Exclusiones
   - Especificar claramente lo que no está incluido
   - Evitar ambigüedades

7. Condiciones comerciales
   - Forma de pago (transferencia bancaria, cheque, etc.)
   - Anticipos (si aplica, sugerir un 30% para cubrir costos iniciales)
   - Plazos de pago (ejemplo: 30 días calendario)
   - Garantías (detallar políticas de garantía si aplica)

8. Validez de la oferta
   - Indicar el período de validez (ejemplo: 30 días desde la emisión)

### Recomendaciones para optimizar costos
- Utilizar herramientas gratuitas como Google Docs para crear y enviar cotizaciones.
- Implementar plantillas reutilizables para ahorrar tiempo.
- Considerar software de gestión de clientes (CRM) gratuito como HubSpot para seguimiento de cotizaciones.

- Lenguaje claro
- Sin términos técnicos innecesarios
- Alcance bien definido
- Exclusiones explícitas
