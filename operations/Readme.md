# Operación y roles

Este documento describe cómo se ejecuta el negocio en el día a día y cómo se distribuyen las responsabilidades. Además, se incluyen recomendaciones para optimizar costos y mejorar la eficiencia.

## Director Técnico y Operaciones

Responsable de:
- Ejecución técnica
- Control de calidad
- Definición de procedimientos
- Supervisión en campo

**Recomendaciones:**
- Utilizar herramientas de gestión de calidad como [Trello](https://trello.com) o [Asana](https://asana.com) para optimizar la supervisión.
- Implementar capacitaciones periódicas certificadas en control de calidad.

## Dirección Comercial y Tecnología

Responsable de:
- Cotizaciones
- Relación con clientes
- Seguimiento comercial
- Organización documental

**Recomendaciones:**
- Adoptar un CRM económico como [HubSpot](https://www.hubspot.com) (versión gratuita) para gestionar relaciones con clientes.
- Certificaciones recomendadas: Curso de ventas consultivas (disponible en plataformas como Coursera o Udemy).

## Seguridad y Salud en el Trabajo

Responsable de:
- Prevención de riesgos
- Ergonomía
- Pausas activas
- Documentación SST

**Recomendaciones:**
- Implementar software de SST como [ErgoSuite](https://www.ergosuite.com) para la gestión de riesgos.
- Certificaciones recomendadas: ISO 45001 para sistemas de gestión de seguridad y salud en el trabajo.

## Flujo estándar

1. Levantamiento
2. Cotización
3. Orden de trabajo
4. Ejecución
5. Verificación
6. Acta de entrega
7. Facturación
8. Seguimiento

**Optimización:**
- Automatizar el flujo con herramientas como [Zapier](https://zapier.com) para conectar sistemas y reducir tiempos.
- Utilizar plantillas digitales para actas y facturación, disponibles en herramientas como Google Workspace o Microsoft 365.
