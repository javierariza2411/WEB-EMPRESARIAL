# Flujo operativo del servicio

Este documento describe paso a paso cómo se ejecuta un servicio desde el primer contacto hasta el cierre. Además, se incluyen recomendaciones para optimizar costos y mejorar la eficiencia.

1. Contacto inicial
   - Cliente describe necesidad
   - Se agenda levantamiento

   **Recomendaciones:**
   - Utilizar herramientas de gestión de citas como [Calendly](https://calendly.com) para automatizar la programación.

2. Levantamiento técnico
   - Revisión en sitio
   - Evidencia fotográfica

   **Recomendaciones:**
   - Usar aplicaciones móviles como [CamScanner](https://www.camscanner.com) para digitalizar evidencias.
   - Certificaciones recomendadas: Curso de inspección técnica en plataformas como Udemy.

3. Cotización
   - Elaboración formal
   - Envío al cliente

   **Recomendaciones:**
   - Automatizar cotizaciones con herramientas como [PandaDoc](https://www.pandadoc.com).

4. Aprobación
   - Confirmación escrita

   **Recomendaciones:**
   - Implementar firmas electrónicas con [DocuSign](https://www.docusign.com) para agilizar el proceso.

5. Ejecución
   - Uso de EPP
   - Checklist técnico

   **Recomendaciones:**
   - Gestionar el inventario de EPP con software como [Sortly](https://www.sortly.com).
   - Certificaciones recomendadas: ISO 9001 para gestión de calidad.

6. Verificación
   - Control de calidad
   - Evidencias

   **Recomendaciones:**
   - Utilizar herramientas de control de calidad como [iAuditor](https://safetyculture.com/iauditor/).

7. Acta de entrega
   - Firma de conformidad

   **Recomendaciones:**
   - Digitalizar actas con plantillas en [Google Workspace](https://workspace.google.com).

8. Facturación
   - Envío formal

   **Recomendaciones:**
   - Automatizar facturación con software como [QuickBooks](https://quickbooks.intuit.com).

9. Seguimiento
   - Posible mantenimiento

   **Recomendaciones:**
   - Implementar recordatorios automáticos con [Mailchimp](https://mailchimp.com) para campañas de seguimiento.
