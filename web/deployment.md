# Despliegue de la web corporativa

Este documento explica cómo se publica la web, los costos asociados y las decisiones técnicas tomadas. Además, se incluyen recomendaciones para optimizar costos y mejorar la eficiencia.

## Principios

- Evitar servidores dedicados
- Reducir costos mensuales
- Alta disponibilidad
- Bajo mantenimiento

**Recomendaciones:**
- Utilizar plataformas serverless como [Vercel](https://vercel.com) o [Netlify](https://www.netlify.com) para despliegues gratuitos o de bajo costo.
- Certificaciones recomendadas: Curso de arquitectura serverless en plataformas como Coursera.

Dominio recomendado:
- .com o .com.co

Costo estimado:
- Anual
- Bajo

La web se despliega en plataformas serverless.

Ventajas:
- No requiere administración de servidor
- Escala automáticamente
- Alta seguridad por defecto

Dominio: pago anual
Hosting: plan gratuito o costo muy bajo
Correo corporativo: gratuito o por usuario
