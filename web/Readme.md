# Página web corporativa

Este documento describe el propósito de la página web y su rol dentro del negocio. Además, se incluyen recomendaciones para optimizar costos y mejorar la eficiencia.

## Objetivo

La web no es una tienda ni un canal de ventas masivas.

Su función es:
- Generar confianza empresarial
- Presentar servicios claramente
- Mostrar experiencia
- Facilitar el contacto con empresas

**Recomendaciones:**
- Utilizar herramientas de análisis como [Google Analytics](https://analytics.google.com) para medir el rendimiento del sitio.
- Certificaciones recomendadas: Curso de diseño web responsivo en plataformas como Udemy.

## Enfoque técnico

La web se construye con:
- Arquitectura estática
- Hosting serverless
- Bajo costo operativo
- Alta disponibilidad

No se utiliza servidor dedicado en la fase inicial.

## Estructura del sitio

- Inicio (mensaje claro industrial)
- Quiénes somos
- Servicios
- Portafolio
- Seguridad y Salud en el Trabajo
- Contacto / Cotización
