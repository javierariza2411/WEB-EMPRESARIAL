# Documentación estratégica

Este directorio contiene documentos de decisión y planeación.

## Documentos

- Plan de creación
- Requisitos por rol
- Roadmap de crecimiento
- Decisiones de socios
- Políticas internas

## Gobierno del proyecto

Toda decisión estratégica debe:
1. Documentarse aquí
2. Ser discutida entre socios
3. Reflejarse en la operación

### Costos aproximados (en pesos colombianos - COP)
- Desarrollo de políticas internas: $800,000 - $1,500,000 COP (utilizando consultores independientes)
- Consultoría estratégica: $2,000,000 - $5,000,000 COP (freelancers o pequeñas firmas locales)

### Tecnologías utilizadas
- Herramientas de documentación: Google Docs (gratuito), Notion, Confluence
- Software de colaboración: Microsoft Teams, Slack, WhatsApp Business (opción económica)

### Certificaciones recomendadas
- Cursos básicos en gestión de proyectos (PMP) en plataformas como Coursera o Udemy
- Certificación en metodologías ágiles (Scrum Master) (opcional para proyectos pequeños)
