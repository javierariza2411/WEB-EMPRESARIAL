# Seguridad y Salud en el Trabajo

La seguridad es un componente central del servicio, no un requisito externo. Además, se incluyen recomendaciones para optimizar costos y mejorar la eficiencia.

## Enfoque

- Prevención antes que corrección
- Reducción de riesgos biomecánicos
- Mejora de productividad
- Cumplimiento empresarial

**Recomendaciones:**
- Implementar software de gestión de SST como [ErgoSuite](https://www.ergosuite.com).
- Certificaciones recomendadas: ISO 45001 para sistemas de gestión de seguridad y salud en el trabajo.

## Componentes

- Inducción SST
- Uso de EPP
- Evaluaciones ergonómicas
- Pausas activas
- Registro documental

**Optimización:**
- Automatizar registros con herramientas como [Google Workspace](https://workspace.google.com) o [Microsoft 365](https://www.microsoft.com).
